I. MY CANVAS - SIMPLE CANVAS EDITOR

- Browser‑Based Canvas Editor Built With HTML, CSS, And Vanilla JavaScript.
- Designed To Deliver A Clean Interface, Fast Performance, And User Friendly Experience.

II. FEATURES

- You Can Change The Project Name
- Add Shapes, Texts, Images
- Switch Between Dark/Light Theme Mode
- Upload Images
- Render The Canvas
- Built-In Color Picker Engine
- Built-In Snap Engine
- Built-In Attach Element Engine
